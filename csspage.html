  1. css언어가 필요한 이유
    • 세밀한 디자인을 하기 위해서 별도의 언어를 만들어야한다.
    • 태그를 사용하는 것보다 더 효율성이 높다.
  2. css 주요 문법 정리
    • 선언.효과: 페이지에서 변화시키고 싶은 속성
      대표적 예시로는 color,font,text..등이 있음
    • 선택자: 위의 효과를 적용할 대상
      태그 선택자-클래스 선택자-id 선택자
    • box model: 웹 사이트를 디자인하는데 필수적인 개념
      border,padding,margin,width등을 이용하여 디자인 가능
      div를 이용하여 디자인 영역을 지정할 수 있음
    • 반응형 디자인: 화면의 크기에 맞추어 디자인을 변형하는 기능
      media(min-width:)...
